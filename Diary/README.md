# Diary App


- Date Picker 사용하여 작성 날짜 기록
- Collection View 를 이용해 일기장 보여주기 -> 클릭 -> 수정, 삭제, 즐겨찾기
- Date Formatter 사용
- 즐겨찾기 목록 구현
